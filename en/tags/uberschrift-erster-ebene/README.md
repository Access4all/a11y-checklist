# 🏷️ Überschrift erster Ebene

## Description

Es ist wichtig, dass mind. eine Überschrift erster Ebene existiert (`<h1>`). Es dürfen durchaus auch mehrere existieren (z.B. eine im `<main>` und eine weitere im `<header>` und `<footer>`).
