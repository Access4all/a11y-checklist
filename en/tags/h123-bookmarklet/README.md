# 🏷️ H123 Bookmarklet

## Description

Ein sehr nützliches [Bookmarklet](https://hinderlingvolkart.github.io/h123/), um Überschriften-Strukturen einer gesamten Webseite anzuzeigen.

Vorteile:

- Ist sehr schnell
- Browser-unabhängig

Nachteile:

- Weniger genau als Screenreader (beachtet gewisse "Edge Cases" u.U. nicht)🇩🇪 Currently only available in German.
