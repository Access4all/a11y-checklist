## Description

The contrast ratio of control elements (e.g. text fields, radio buttons, checkboxes, buttons, tabs, etc.) to the surrounding colours is at least â€˜3:1â€™. This applies to all visual cues that are required for perception and operation (e.g. form field boundaries, fold-out indicators for flyouts/dropdowns, tick in a checkbox, etc.), in particular also for the perception of the status of an element. The hover state of an element does not have to be distinguishable from the standard state.

## Method

**Manual check:** Look through elements, interact with them and look for weak contrasts. Contrast determination by Colour Contrast Analyser if necessary.
