# 🏷️ NerdeFocus Extension

## Beschreibung

Zeichnet einen dicken Rand um das aktuell fokussierte Element. Praktisch z.B. bei einem Fokus-Reset ([🏷️ Fokus-Reset (Design)](/de/tags/fokus-reset-design)).
