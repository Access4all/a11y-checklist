# 🏷️ Korrekte Links und Schalter

## Beschreibung

Link verwendet man, wenn man auf eine neue Seite zeigt. Button nimmt man, wenn innerhalb der aktuellen Seite etwas passieren soll (oder man ein Formular abschicken soll).
