# 🏷️ Fokussierbarkeit

## Beschreibung

Nativ interaktive HTML-Elemente sind immer fokussierbar, z.B. `<input>`, `<select>`, `<button>`, `<a href>` (wichtig: ohne `href` ist es nicht fokussierbar), etc.

Zu `tabindex="0"` raten wir nur in einigen wenigen Sonderfällen.
